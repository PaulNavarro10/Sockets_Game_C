Importante: Requisito, tener compilador GCC
1) Clonar el repositorio https://github.com/PaulNavarro10/Sockets_Game_C.git
2) En la terminal ejecutar gcc auxiliar_funciones_9.c bot_grupo_9.c func_grupo__9.c Globales_grupo_9.c main_grupo_9.c mensaje_funciones_9.c menu_funciones_9.c partida_funciones_9.c -o bot9 -lws2_32
3) En la terminal ejecutar bot9.exe, configurar Modo local y especificar la ruta donde se guardara el proceso del juego en .txt
4) En la terminal ejecutar bot9.exe, configurar Modo Visita y especificar la ruta donde se guardara el proceso del juego en .txt
5) Dar empezar partida para el Local.
6) Dar empezar partida para el Visitante.
